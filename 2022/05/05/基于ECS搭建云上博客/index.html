基于ECS搭建FTP服务

  1. 安装vsftpd
    • 在Web Terminal输入：yum install -y vsftpd
  2. 开启FTP服务开机自启动
    • systemctl enable vsftpd.service
  3. 启动FTP
    • systemctl start vsftpd.service

使用云助手自动化运维服务器ECS实例

  1. 在云服务器控制台，选择发送命令云助手
    • detect_linux_release_type.sh

安装Apache

  1. 打开Web Terminal
    • 输入指令 yum -y install httpd mod_ssl mod_perl mod_auth_mysql
    • 检查是否安装成功：httpd -v
    • 启动Apache服务：systemctl start httpd.service
    • 远程桌面
    • 在浏览器访问http://<ECS公网地址>

    • 修改WordPress配置文件

      • 进入/usr/share/wordpress目录。

      • 查看修改后的目录结构
